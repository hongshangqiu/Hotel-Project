# 易宿酒店预订平台 (Easy Stay)

本项目为 **第五期前端训练营** 大作业。

## 项目信息

*   **项目名称**：易宿酒店预订平台
*   **团队人数**：2-3 人
*   **截止时间**：2026 年 2 月 26 日 20:00

## 核心功能

本项目包含两个核心端：

1.  **用户端 (移动端)**：面向消费者，提供酒店查询、列表浏览和详情查看功能。
2.  **管理端 (PC 站点)**：面向商户（录入信息）和管理员（审核发布），实现酒店信息的全生命周期管理。

## 技术栈

*   **前端框架**：Taro 3 (React + Webpack 5)
*   **开发语言**：TypeScript (TS)
*   **状态管理**：Zustand
*   **UI 组件库**：NutUI-React
*   **后端 (模拟)**：Node.js (待定)

## 快速启动

```bash
# 安装依赖
npm install

# 启动 H5 开发环境 (推荐用于调试 PC/移动端响应式)
npm run dev:h5

# 启动微信小程序开发环境
npm run dev:weapp
```

## 目录结构

```text
src/
├── shared/       # 共享逻辑 (类型、状态、接口)
│   ├── store/    # Zustand 状态管理
│   ├── types/    # TypeScript 接口定义
│   └── services/ # API 请求封装
├── pages/        # 视图层 (分端适配)
│   ├── user/     # 用户端 (移动端布局)
│   │   ├── index # 酒店查询页 (首页)
│   │   ├── list  # 酒店列表页
│   │   └── detail# 酒店详情页
│   └── admin/    # 管理端 (PC端布局)
│       ├── login # 登录/注册
│       ├── manage# 信息录入/编辑
│       └── audit # 审核/发布列表
```
