# 易宿酒店预订平台 (Easy Stay)

> 本项目为 **第五期前端训练营** 大作业，一个基于 Taro 3 + React 的全端酒店预订管理系统。

## 项目简介

易宿酒店预订平台是一个支持多端（PC端管理后台 + 移动端用户前台）的酒店管理系统。采用 Taro 3 框架开发，可同时编译为 H5 网页端和微信小程序。

**核心功能：**
- 🏨 酒店查询与预订（用户端）
- 🏢 酒店信息录入与管理（商户端）
- ✅ 酒店审核与发布（管理员端）
- 📱 响应式布局，PC端与移动端自动适配

## 技术栈

- **前端框架**：Taro 3 (React + Webpack 5)
- **开发语言**：TypeScript (TS)
- **状态管理**：Zustand
- **UI 组件库**：NutUI-React
- **CSS 方案**：Sass/SCSS

## 快速启动

```bash
# 安装依赖
npm install

# 启动 H5 开发环境 (网页端预览)
npm run dev:h5

# 启动微信小程序开发环境
npm run dev:weapp

# 构建生产环境
npm run build:h5
```

## 项目目录结构

```
src/
├── app.tsx                    # 应用入口，登录页不包裹Layout
├── app.config.ts              # 页面路由配置
├── app.scss                   # 全局样式
├── components/                # 公共组件
│   ├── Layout/               # 响应式布局组件
│   ├── StarRating/           # 星级评分组件
│   └── Calendar/             # 日历组件
├── pages/                    # 页面
│   ├── user/                # 用户端 (移动端布局)
│   │   ├── index/           # 首页 (酒店查询)
│   │   ├── list/            # 酒店列表
│   │   └── detail/          # 酒店详情
│   └── admin/               # 管理端 (PC端布局)
│       ├── login/           # 登录页
│       ├── register/       # 注册页
│       ├── manage/          # 商户录入页
│       └── audit/           # 审核页
└── shared/                  # 共享逻辑
    ├── store/               # 状态管理
    ├── types/               # 类型定义
    ├── services/            # 服务层 (Mock API)
    └── utils/               # 工具函数
```

## 注意事项

1. `@/` 别名指向 `src/` 目录
2. H5端访问 `http://localhost:10086`
3. 微信小程序需使用微信开发者工具打开 `dist/` 目录
